Working With This Theme

Let's start with Jonroc custom classes and common styles/techniques. We use rem for most sizing which helps keep things uniform and also plays nice with Bootstrap. All theme related styles should go in theme.scss and we use Bootstrap variables and css in our custom work as much as possible (the css is already there might as well use it). However, many times when it comes to padding or margins Bootstrap classes can make the code cluttered and hard to read, so often we create our own classes to deal with padding and margin unless there is only one or two bootstrap classes needed to give it padding or margin (same thing with FLEX most times it's better to put it into a non-bootstrap class). It's also best to create a custom class when the item is used on more than one page or for items that repeat on page so that it's easier to mass update styles across a page or the entire site. 

The content below is for conviently styling Bootstrap only. Use custom.scss in the bootstrap theme folder to customize the varaibles. We never directly change the boostrap.css, to keep things working well we only use Bootstrap variables or override Bootstrap in custom.scss if you can't find a variable.

We'll get the most common boostrap elements on this page so you can check your styling, remember WordPress may overwrite some of this.
